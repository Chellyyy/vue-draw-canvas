# vue-draw-canvas

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Quick Start
```
import Vue from 'vue'
import DrawCanvas from 'vue-draw-canvas'

Vue.use(DrawCanvas)

// or
import { DrawCanvase } from 'vue-draw-canvas'

Vue.component(DrawCanvase.name, DrawCanvase)
```
